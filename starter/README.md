# Employee Polls Application

This project is a React/Redux application that allows employees to create and answer "Would You Rather" polls.

## Installation

1.  Download the source code.
2.  Navigate to the project directory:
    ```bash
    cd starter
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```

To launch the application in development mode, run:

```bash
npm start
```

The app will be available at `http://localhost:5173`.

### Run Tests

To execute the unit tests, run:

```bash
npm test
```

The project includes more than 10 tests covering data functions, components, and logic.

## Features

- Login/Logout system with user impersonation.
- Homepage with Answered and Unanswered poll categories.
- Poll details page with voting and results.
- Leaderboard showing user engagement scores.
- "Add New Poll" form.
